# Ezenwa Joseph Chibuzo — Frontend Developer Portfolio

> A professional, multi-page frontend developer portfolio built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

## 🌐 Live Demo
[View Portfolio →](https://josephchibuzo-dev.github.io/portfolio)

## 📁 Pages
| Page | Description |
|------|-------------|
| `index.html` | Hero, featured projects, stats, CTA |
| `about.html` | Story, timeline, values |
| `projects.html` | Full project showcase with filtering |
| `skills.html` | Animated skill bars, tech stack, soft skills |
| `contact.html` | Contact form, social links, availability |

## 🛠 Tech Stack
- **HTML5** — Semantic markup
- **CSS3** — Custom properties, animations, Grid/Flexbox, responsive design
- **Vanilla JavaScript** — Typewriter effect, counter animations, scroll reveal, filter logic, form handling

## ✨ Features
- 🎨 Bold & colorful design with custom CSS variables
- 🖱 Custom animated cursor
- ⌨️ Typewriter role animation on hero
- 📊 Animated skill progress bars (triggered on scroll)
- 🔍 Project filtering by category
- 📱 Fully responsive (mobile-first)
- ♿ Accessible & semantic HTML
- 🌀 Scroll-triggered reveal animations
- 💾 No external JS dependencies

## 🚀 Deploy to GitHub Pages
1. Fork or clone this repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Your site will be live at `https://josephchibuzo-dev.github.io/repo-name`

## 📝 Customization
- Update personal info in each HTML file
- Replace placeholder links with your real GitHub, LinkedIn, Twitter
- Add your real email in `contact.html`
- Add actual project screenshots/links in `projects.html`
- Upload your photo and replace the initials placeholder in `about.html`
